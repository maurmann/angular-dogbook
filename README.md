# angular-dogbook
Versão para amantes de cachorros da aplicação gatitobook da Alura

# Criação do Projeto
ng new dogbook --strict
- utilizar routing: y
- CSS/SCSS/etc: CSS

# Instalação biblioteca CSS Bootstrap e Fontawesome para icones
npm install bootstrap font-awesome
Adicionar as referencias no arquivo angular.json, na raiz do projeto

"styles": [
              "src/styles.css",
              "./node_modules/bootstrap/dist/css/bootstrap.min.css",
              "./node_modules/font-awesome/css/font-awesome.css"
            ]
